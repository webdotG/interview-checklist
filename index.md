Описание компонентов

1. appInitializer.js
    Отвечает за:
    Начальную загрузку приложения. Определяет, на какой странице находится пользователь (index.html или interviews.html), и запускает соответствующий главный компонент (InterviewManager).

2. InterviewManager.js
    Отвечает за:
    Центральное управление страницей. Координирует работу всех других компонентов, обрабатывает действия пользователя (например, сохранение), управляет процессом сохранения данных в Firebase и локально.

3. UIManager.js
    Отвечает за:
    Управление интерфейсом. Отображает сообщения о сохранении, ошибки, сообщения об успехе, управляет видимостью элементов UI. В этом модуле он также может управлять состоянием кнопок (например, "Сохранить").

4. AuthUI.js
    Отвечает за:
    Всю логику, связанную с авторизацией. Управляет кнопками входа/выхода, отображает информацию о пользователе и предупреждения. Использует auth.service.js для взаимодействия с Firebase Authentication.

5. InterviewForm.js
    Отвечает за:
    Создание и управление динамической формой. Генерирует HTML-разметку для вопросов на основе данных из questions.data.js, собирает ответы пользователя и предоставляет методы для валидации и получения данных формы.

6. db.service.js
    Отвечает за:
    Взаимодействие с базой данных Firebase. Предоставляет InterviewManager методы для сохранения новых интервью.

7. auth.service.js
    Отвечает за:
    Всю логику авторизации с Firebase. Предоставляет AuthUI методы для входа, выхода и получения текущего пользователя.

8. Вспомогательные сервисы
    notification.service.js:
    предоставляет механизм для отображения временных сообщений (успех, ошибка, информация) пользователю.
    form.validator.js: содержит правила и логику для проверки полей формы.
    questions.data.js: статические данные с вопросами и их структурой.
